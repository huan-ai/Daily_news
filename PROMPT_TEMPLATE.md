# AI 日报自动生成系统 —— 核心复刻 Prompt (Version 1.0)

> **使用说明**：以后如果在其他项目或给其他 AI 助手布置类似任务时，直接把下面这段完整的 Prompt 发送给它，就可以一次性得到我们经过多次迭代后才成熟的系统架构和代码实现。

## 🎯 系统核心复刻 Prompt

```text
你好！我需要你帮我从零到一开发一个【AI 行业日报自动生成系统】。这个系统需要每天自动收集 AI 领域的最新动态，利用大模型（LLM）进行深度分析，并将最终的高质量 Markdown 报告通过邮件发送给我。

请详细阅读以下需求说明，并按模块为我提供完整的代码实现、配置文件以及使用文档。

### 1. 核心目标与功能需求
- **自动化运行**：完全无人值守，使用定时任务（如 macOS LaunchAgent 或 crontab）实现每天固定时间（如 17:30）准时触发。
- **多源数据抓取**：
  - **GitHub 热榜**：抓取每日 Top 5 的 AI 相关开源项目。不仅要抓取名称、Star 数和描述，**必须通过调用 GitHub API 获取每个仓库的 README 摘要**，作为上下文提供给大模型。
  - **RSS/第三方接口**：抓取 Hacker News 等权威渠道的每日最热 AI 新闻或技术文章。
- **大模型（LLM）深度处理**：
  - 接驳先进的大模型（如 Gemini 2.5 Flash 或 GPT-4 等）。
  - 对收集到的原始素材进行：去重、打分、分类（分为：大模型进展、开源动态、多模态突破、智能体生态等）。
  - **输出要求**：对每个 GitHub 项目和新闻，利用大语言模型生成 100-200 字的**深度中文介绍和分析**（避免干瘪的属性罗列，需包含"是什么"、"解决什么问题"、"技术亮点"和"适用场景"）。除英文专有名词外，报告整体必须是高质量的中文。
- **多格式输出与推送**：
  - 自动将大模型的分析结果组合生成符合排版规范的 Markdown 格式日志（适合直接复制在公众号或小红书发表），同时保留一份纯文本版备用。
  - 使用 SMTP 协议将这份 Markdown 渲染出来的报告自动发送到多个指定邮箱（如 163 邮箱发送到多个 QQ 邮箱）。

### 2. 架构设计与安全规范要求
- **项目结构分离**：请采用标准的 Python 应用工程结构（分离出 `src/collectors`、`src/processors`、`src/generators`、`src/utils` 等子目录，配合 `config/` 放配置文件，`data/reports/` 放输出文件）。
- **零明文密钥（极其重要）**：
  - API Key（如 Gemini Key）和邮箱密码授权码**绝对不能**硬编码在代码或持久化的普通配置文件中。
  - 对于 macOS 部署，要求提供从 Keychain 中安全读取这些密钥的方法封装（或者使用安全启动脚本注入环境变量的方式）。
  - 提供 `.env.example`，并且一定要配置好 `.gitignore` 以排除敏感信息及日志输出文件。

### 3. 具体实现指引请求
请根据上述要求，为我提供一份全面的 **实现计划 (Implementation Plan)**，并给出以下核心模块的代码：
1. **主配置系统**：设计 `config.yaml` 格式体系。
2. **收集与分析**：重点给出带 README 抓取的 `github_collector.py` 以及大模型驱动的 `analyzer.py` 的关键逻辑。
3. **安全启动流**：编写一个能安全加载凭据然后执行 Python 入口的 Shell 脚本思路（如 `run_secure.sh`）。
4. **README 文档**：给出一份非常专业的项目整体说明及部署指南。
```

## 💡 为什么这份 Prompt 很强大？

这份 Prompt 凝结了我们在项目中踩过的几个大坑并提前做出了约束：

1. **提前规避了内容干瘪的坑**：明确要求必须抓取 GitHub 仓库的 **README**，并强制大模型输出 100-200 字包含具体技术亮点的中文介绍，避免了早期版本只列出 "项目名+Stars" 的贫乏结果。
2. **提前设定了安全与敏感信息保护**：明确指出了**零明文密钥**和利用环境变量 / Keychain 注入的要求，避免了以后重写定时任务和迁移代码时面临的安全风险。
3. **良好的工程结构**：一上来就要求分离 Collector、Processor、Generator 和 Config，保证了这套代码具备很高的数据源拓展性（未来可以很轻松地加 Twitter或微信公众号的爬虫进去）。
